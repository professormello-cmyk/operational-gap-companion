# Operational Gap Companion

Operational Gap Companion is a lightweight web application for operational-gap diagnostics in inorganic electronic-state assignment.

The app is designed to support quick inspection of local state competition, near-degeneracy corridors, two-state mixing, and assignment robustness under chemically relevant control variables.

## Main purpose

The main purpose of the app is to provide a simple and usable diagnostic companion for:

- operational-gap analysis
- 2×2 state-mixing inspection
- chemically sensitive assignment changes
- element-based preset loading from the periodic table

## Current and planned features

- clickable periodic table in standard layout
- editable diagnostic presets for each element
- 2×2 calculator using Δ, V, Δop, and T
- short diagnostic interpretation in English
- lightweight static deployment
- offline support through Progressive Web App behaviour

## Scope

The app is intended as a compact technical companion for inorganic chemistry, especially in cases involving:

- spin-state competition
- charge-transfer competition
- relativistic and spin-orbit-sensitive reordering
- operationally fragile electronic-state assignments

The app does not replace full electronic-structure calculations. It is intended as a rapid interpretive and diagnostic layer.
