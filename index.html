Operational Gap Companion

Pocket Corridor Table + 2×2 diagnostic calculator for inorganic electronic-state assignment

1) 2×2 diagnostic calculator

Formulae (audit view) 
R = |Δ|/|V|
sin²φ = 1/2 * (1 - Δ/sqrt(Δ² + 4V²))
Δadiab = sqrt(Δ² + 4V²)
Δmin = 2|V|
kBT (eV) = 8.617333262e-5 * T
CRS thresholds: CRS=0 if Δop≥0.5 and R≥5; CRS=1 if Δop≥0.2 and R≥2; CRS=2 if Δop≥0.1 and R≥1; otherwise CRS=3
