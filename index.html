Pocket Corridor Table (sd/df)

PCPT + corridor calculator (Δ, V, Δop → R, sin²φ, CRS)

1) 2×2 calculator (from the paper)

Formulae (for audit) 
R = |Δ|/|V|
sin²φ = 1/2 * (1 - Δ/sqrt(Δ² + 4V²))
Δmix = sqrt(Δ² + 4V²)
kBT (eV) = 8.617333262e-5 * T
CRS heuristic (editable): CRS=0 if Δop≥0.5 and R≥5; CRS=1 if Δop≥0.2 and R≥2; CRS=2 if Δop≥0.1 and R≥1; otherwise CRS=3

2) PCPT by element (CSV)

File: data/pcpt.csv (symbol, CRS, DMC, note). The table below reads this CSV.

3) Cases dataset (syntheses/complexes) — optional

File: data/cases.csv with (Δ, V, Δop, θ, observable) for attaching real cases and generating reports.

Offline: this site registers a Service Worker when served over HTTPS (Netlify/GitHub Pages). If opened as a local file (file://), the PWA cannot be installed.
